### Clone the project

#### SSH 
`git clone git@github.com:ak-shumkar/eduwise-backend.git`

#### HTTPS
`git clone https://github.com/ak-shumkar/eduwise-backend.git`


### Build the project

#### Build
`docker-compose up -d --build`

#### Check status
`docker-compose ps`
